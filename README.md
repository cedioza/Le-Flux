# Le Flux

_Un constructor visual nativo de flujos y orquestador de agentes para Mistral AI._

"Le Flux" lleva la elegancia de "La Plateforme" de Mistral a un canvas visual interactivo por nodos. Permite a los desarrolladores orquestar flujos de IA complejos, integrar herramientas (RAG, Búsqueda Web, Ejecución de Código) y delegar tareas entre modelos (como Pixtral 12B) de manera fluida.

## Características
- **Editor Visual de Nodos**: Desarrollado con React Flow para una experiencia interactiva Drag-and-Drop.
- **UI Nativa Estilo Mistral**: Diseñado con una estética de "dev-tool" en modo oscuro, replicando la experiencia premium de Mistral AI.
- **Panel de Logs en Tiempo Real**: Seguimiento de ejecución en vivo, métricas de tokens y latencia.
- **Nodos Especializados**: Configuración de System Prompts, selección de Modelos Mistral y herramientas directamente desde la UI.

## Inicio Rápido

Para correr la aplicación localmente:

```bash
npm install
npm run dev
```

El proyecto estará disponible en `http://localhost:5173`.
